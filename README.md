# Budget Analyzer

Automated personal/business budget analyzer that processes expense data, 
categorizes spending, calculates savings rates and generates visual reports.

## What it does
- Processes monthly income and expense data from Excel
- Categorizes spending automatically
- Calculates savings rate per month and classifies as Excellent/Regular/Critical
- Generates 4 visualizations: spending by category, distribution, income vs expense, savings rate

## Tech Stack
- **Python** — automation
- **Pandas** — data analysis
- **Matplotlib & Seaborn** — visualizations
- **Git/GitHub** — version control

## Run
```bash
python scripts/run_all.py
```
